# Cyber Crime Complaint Dashboard

A web-based dashboard for recording and visualizing cyber crime complaints.

## Features

- Record new cyber crime complaints with details like type, description, location, and date
- Interactive visualizations showing:
  - Crime trends over time
  - Distribution by crime type
  - Geographical distribution
  - Time analysis of when crimes are reported
- View raw data in tabular format
- Responsive design that works on desktop and mobile devices
